# Addressable LED Delay Splitter (PCB)

This folder contains the [KiCad](https://kicad.org/) project for the Addressable LED Delay Splitter PCB.

## Production files

Production files can be found in the `jlcpcb/production_files` folder.
These files were generated by the [KiCAD JLCPCB tools plugin](https://github.com/bouni/kicad-jlcpcb-tools).

It should be possible to upload the GERBER archive to other PCB manufacturers as well, but this has not been tested.

The BOM and CPL files are in the format required by [JLCPCB](https://jlcpcb.com/).

It may be required to check and adjust the part numbers before ordering, because parts are from time to time replaced or go out of stock.
